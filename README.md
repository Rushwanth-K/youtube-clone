# 🎬 YouTube Clone

A YouTube homepage clone built using HTML and CSS. This project replicates the core UI of YouTube including the header, sidebar, and video grid layout.

---

## 🚀 Live Demo

https://rushwanth-k.github.io/youtube-clone/

---

## 💻 GitHub Repository

https://github.com/Rushwanth-K/youtube-clone

---

## ✨ Features

- YouTube-style header with search bar  
- Sidebar navigation menu  
- Video grid layout  
- Thumbnails with video duration  
- Channel profile section  
- Notification icon with count  
- Clean and structured UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- Google Fonts (Roboto)
